The DL-Media System (see also latest changes)

DL-Media is ontology mediated multimedia information retrieval system, which combines logic-based retrieval with multimedia feature-based similarity retrieval. An ontology layer may be used to define (in terms of a DLR-Lite like description logic) the relevant abstract concepts and relations of the application domain, while a content-based multimedia retrieval system is used for feature-based retrieval.

DL-Media's most interesting features are:

  • Fuzzy DLR-Lite: as logical language for schema/ontology specification
  • Fuzzy Conjunctive Queries: as query language
  • Relational/Multimedia Database: data may be stored into relational database and/or multimedia database
  • Top-k Query Answering: Supports top-k retrieval (find top-k scored tuples satisfying query)
