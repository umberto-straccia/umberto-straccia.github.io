This directory contains:

README.txt: this file;

FOILDL.jar: implementation of the GUI part of FOILDL;

lib: directory containing the libraries needed by FOILDL, among these jar archives
there are FOILDL.foil.jar containing the interface defined by FOILDL; 
FOILDL.foil.owlapi.jar contains implementation of programming interface of FOILDL.foil.jar using OWL API;
other jar archives for OWL API and Pellet, Hermit and Jfact are organized in subdirectories;

FOILDL.foil.protege.jar: plugin version of FOILDL for the Protege editor;

examples: OWL ontologies used as test-suit are stored here;

CONFIG: default configuration file containing examples of CONFIG usage;

USER_MANUAL: instructions to use FOILDL as stand-alone application or as Protege plugin;
