Description Logic Programs under Probabilistic Uncertainty and Fuzzy Vagueness

In 9th European Conferences on Symbolic and Quantitative Approaches to Reasoning with Uncertainty (ECSQARU-07).

Abstract:
Towards an infrastructure for handling uncertainty and vagueness in the reasoning layers of the Semantic Web, we present an approach to description logic programs (with stratified negation) that allow for dealing with probabilistic uncertainty as well as fuzzy vagueness. We also provide algorithms for query processing in such programs. Furthermore, we describe a shopping agent example, which gives evidence of their usefulness in realistic web applications.
