Managing Uncertainty and Vagueness in Description Logics for the Semantic Web

In Journal of Web Semantics, 2008

Abstract:
Ontologies play a crucial role in the development of the SemanticWeb as a means for defining shared terms in web resources. They are formulated in web ontology languages, which are based on expressive description logics. Significant research efforts in the semantic web community are recently directed towards representing and reasoning with uncertainty and vagueness in ontologies for the Semantic Web. In this paper, we give an overview of approaches in this context to managing probabilistic uncertainty, possibilistic uncertainty, and vagueness in expressive description logics for the Semantic Web.
