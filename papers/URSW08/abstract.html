DLMedia: an Ontology Mediated Multimedia Information Retrieval System

In International Workshop on Uncertainty Reasoning for the Semantic Web (URSW-08).

Abstract:
We outline DLMedia, an ontology mediated multimedia information retrieval system, which combines logic-based retrieval with multimedia feature-based similarity retrieval. An ontology layer may be used to define (in terms of a DLR-Lite like description logic) the relevant abstract concepts and relations of the application domain, while a content-based multimedia retrieval system is used for feature-based retrieval.
